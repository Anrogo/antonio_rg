A continuación te ofrecemos la lista de los lenguajes de programación más populares de 2019, entre los que se encuentran algunos tan conocidos como Python, Java, C, entre otros. Esto nos puede dar una pista de aquellos que cuentan con mayores integrantes en la comunidad, algo que puede ser en cierto aspecto muy positivo, pero también negativo si nos movemos dentro de un nicho de mercado con pocas ofertas de empleo. Ahora la IEEE Spectrum ha publicado su lista anual de los principales lenguajes de programación de 2019, donde hay pocas sorpresas, dado que Python, que ya aparece primero en multitud de otras tantas listas, copa el primer lugar.

SQL, Structure Query Language (Lenguaje de Consulta Estructurado) es un lenguaje de programacion para trabajar con base de datos relacionales como MySQL, Oracle, etc.

MySQL es un interpretador de SQL, es un servidor de base de datos.

MySQL permite crear base de datos y tablas, insertar datos, modificarlos, eliminarlos, ordenarlos, hacer consultas y realizar muchas operaciones, etc., resumiendo: administrar bases de datos.

Ingresando instrucciones en la linea de comandos o embebidas en un lenguaje como PHP nos comunicamos con el servidor. Cada sentencia debe acabar con punto y coma (;).
